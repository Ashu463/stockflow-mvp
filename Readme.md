# StockFlow MVP

A minimal SaaS inventory management system built in 6 hours.

## Stack

Backend
- NestJS
- Prisma
- SQLite

Frontend
- React
- Vite

## Features

- Authentication
- Product CRUD
- Inventory dashboard
- Low stock alerts
